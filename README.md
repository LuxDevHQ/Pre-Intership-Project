### May/June Pre-Intership Project:

Build a healthcare analytics system using a dataset from Kaggle. The system should clean and store the data in a PostgreSQL database and retrain a machine learning model every Saturday at 12:00 noon to predict patient test results as Normal, Abnormal, or Inconclusive. Deploy the model using Flask or FastAPI, push your project to GitHub, host it on a free platform such as Vercel, and share the live API link.
